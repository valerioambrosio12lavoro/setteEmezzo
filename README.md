## Descrizione
Questo progetto è una versione digitale e interattiva del gioco di carte "Sette e Mezzo". Sviluppato in Java, il software non si limita alla logica di gioco, ma integra un'interfaccia grafica completa e un sistema di gestione della partita dinamico.

L'obiettivo principale dell'esercizio è dimostrare la padronanza dei pilastri della programmazione a oggetti, con un focus particolare sulla gestione del flusso di gioco e sulla persistenza dei dati.

## Funzionalità
Interfaccia Grafica (GUI): Implementazione di una finestra di gioco interattiva tramite la libreria Swing, con gestione in tempo reale di eventi, bottoni e aggiornamenti visuali.

Logica Avanzata (OOP): Utilizzo di Ereditarietà e Polimorfismo per gestire diverse modalità di gioco (standard e "truccata") senza riscrivere il codice di base.

Persistenza Dati (I/O): Salvataggio automatico del log delle partite e dei risultati finali su un file esterno (storico.txt) per il monitoraggio delle statistiche.

Gestione Eccezioni: Robustezza del codice garantita da un sistema di gestione degli errori (try-catch) per le operazioni di input/output e per la validazione delle scelte dell'utente.

## Tecnologie utilizzate
Linguaggio: Java

Librerie Grafiche: javax.swing (Swing) e java.awt

Persistenza: java.io (File Writing)

Strumenti: Java Collections Framework (ArrayList)
