Una versione digitale del classico gioco di carte "Sette e Mezzo", sviluppata in Java. Il progetto include un'interfaccia grafica (GUI) interattiva e una logica di gioco avanzata che mette in pratica i pilastri della programmazione a oggetti.

Il software include una funzionalità di "Partita Truccata" attivata casualmente dal sistema, che dimostra l'uso del Polimorfismo per cambiare il comportamento del mazzo durante l'esecuzione (Runtime).
